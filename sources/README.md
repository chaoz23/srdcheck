# Sources & Provenance

All rule content in this repository derives from exactly one document:

| Document | URL | SHA-256 |
|---|---|---|
| System Reference Document 5.2.1 (CC-BY-4.0) | https://media.dndbeyond.com/compendium-images/srd/5.2/SRD_CC_v5.2.1.pdf | see `SRD_CC_v5.2.1.pdf.sha256` |
